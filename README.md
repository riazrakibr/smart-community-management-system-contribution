# Smart Community Management System

A MERN Stack based web application for managing community issues efficiently.

🚀 **Live Demo:**
https://smart-community-management-system-1.onrender.com

---

## Features

* User Registration & Login
* JWT Authentication
* Admin and Resident Roles
* Create Community Issues
* Upload Issue Images
* Update Issue Status
* Delete Issues
* Search and Filter Issues
* Responsive Dashboard
* MongoDB Atlas Integration
* Deployed on Render

---

## Tech Stack

### Frontend

* React.js
* Axios
* React Router DOM
* CSS

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT Authentication
* Multer

---

## Installation

### Clone Repository

```bash
git clone https://github.com/Pavanclg/smart-community-management-system.git
```

---

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

### Backend Setup

```bash
cd server
npm install
npm start
```

---

## Environment Variables

Create `.env` file inside server folder:

```env
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
```

---

## Deployment

* Frontend Hosted on Render Static Site
* Backend Hosted on Render Web Service
* Database Hosted on MongoDB Atlas
